# AI Fashion Outfit Recommender

An image-based fashion recommendation system that suggests visually compatible outfit items using deep learning and similarity search.

## Overview
This project uses a convolutional neural network to extract visual embeddings from clothing images and retrieves compatible outfit items using nearest-neighbor search.

## Features
- ResNet-50 image embeddings
- FAISS similarity search
- Rule-based outfit compatibility
- Visual recommendation grids

## Tech Stack
Python, PyTorch, FAISS, Google Colab

## Data
Fashion Product Images dataset from Kaggle. Dataset not included in this repository.

Open the notebook in Google Colab and run the cells in order.
